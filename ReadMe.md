# SkillSwap

SkillSwap is a student-focused skill exchange platform where users can connect, teach skills they know, and learn new skills from others. The platform helps students collaborate and grow by exchanging knowledge in different domains.

## Features

* Student skill exchange platform
* User profile section
* Skills to Teach & Skills to Learn
* Responsive design
* Backend integration with MongoDB
* Form data storage

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/yourusername/SkillSwap.git
```

2. Go to project folder

```bash
cd SkillSwap
```

3. Install dependencies

```bash
cd backend
npm install
```

4. Start server

```bash
npx nodemon server.js
```
